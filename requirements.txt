# Option 1: PyMuPDF (fitz) - RECOMMENDED in PROMPT.md
# Pros: Fast, memory-efficient, supports text/image extraction, annotations, rendering
# Cons: C dependencies (complex install), commercial license for some uses, steeper learning curve
# Best for: High-performance text extraction, large files
# PyMuPDF==1.23.8

# Option 2: pypdf (formerly PyPDF2)
# Pros: Pure Python, easy install, simple API, actively maintained
# Cons: Slower on large files, limited advanced features
# Best for: Simple text extraction, basic operations
# pypdf==4.0.1

# Option 3: pdfplumber
# Pros: Excellent for tables/structured data, precise text extraction, good layout analysis
# Cons: Slower than PyMuPDF, heavier dependencies
# Best for: Complex layouts, table extraction
# pdfplumber==0.10.3

# Option 4: pdfminer.six
# Pros: Fine-grained control, handles complex layouts well
# Cons: Complex API, slower performance, steeper learning curve
# Best for: Advanced parsing needs
# pdfminer.six==20221105
